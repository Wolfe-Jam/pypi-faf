# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, give people the benefit of the doubt, focus on the work. Harassment, personal attacks, and other unwelcoming behavior are not OK and will be addressed.

**Report a concern:** open a [private security advisory](https://github.com/Wolfe-Jam/pypi-faf/security/advisories/new) or email the maintainer via the GitHub profile.

The full Covenant text is canonical — read it at the link above.
